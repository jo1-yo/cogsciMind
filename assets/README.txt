ASSETS FOLDER
=============

This folder is for website resources such as:
- Video thumbnails
- Images
- Icons
- Other media files

Video Thumbnail:
- Filename: main-thumbnail.jpg
- Used as poster image for the video player
- Recommended size: 1280x720 pixels
- Supported formats: JPG, PNG, WebP

Tips:
- Thumbnails can be screenshots from your video
- Or custom-designed cover images
- Optimize images for web to reduce file size

The thumbnail is optional. If not provided, the video player will show 
a black screen until the video starts playing.
